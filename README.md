# Desafio imagem

### Check list

- [x] O stdout mostrar  "Full Cycle Rocks!!"
- [x] Imagem com menos de 2mb

## Buildar imagem

```shell
  # para o build da imagem execute 
  chmod +x build.sh
  # em seguida
  docker build -t fullcycle .
  # para executar a imagem
  docker run fullcyle
  # stdout "Full Cycle Rocks!!"
```
